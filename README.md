# NIGHT - Nginx Integrated Guard & Hardening Toolkit

Automated Nginx configuration auditing, hardening, and integrated defense toolkit for web server security.

---

## Features

- Configuration auditing (detect misconfigurations, CWE-aligned issues)
- Automated hardening (secure config generation)
- Active protection (ModSecurity, UFW, Fail2ban integration)
- Security testing (Nikto, Gobuster, etc.)
- Comparative reporting (before vs after hardening)

---

## Architecture

- Passive protection (audit + harden)
- Active protection (WAF, IPS, firewall)
- Security testing & evaluation
